ARSITEKTUR APLIKASI: KOPISERU MARKETING DASHBOARD
=================================================

1. RINGKASAN SISTEM
-------------------
KopiSeru Marketing Dashboard adalah aplikasi berbasis web satu halaman (one-screen dashboard) yang dibuat menggunakan Streamlit untuk memvisualisasikan data pemasaran, transaksi, dan pendapatan KopiSeru secara interaktif. Desain antarmuka dioptimalkan agar pas di satu layar tanpa scrollbar vertikal pada resolusi laptop standar.

2. TEKNOLOGI YANG DIGUNAKAN
---------------------------
- Bahasa Pemrograman: Python 3.12+
- Web Framework: Streamlit (UI & Reactive Server)
- Analisis Data: Pandas, NumPy
- Visualisasi Data: Plotly (Interactive Graphs)
- Pembaca Excel: OpenPyXL

3. LAPISAN DATA (DATA LAYER)
----------------------------
- Sumber Data: File Excel `coffe_shop_final.xlsx` (atau file serupa di folder proyek).
- Mekanisme Pencarian File:
  * Mengecek environment variable `KOPISERU_DATA_PATH`.
  * Mencari beberapa nama file kandidat default (misal `coffe_shop_final.xlsx`).
  * Fallback otomatis ke file `.xlsx` pertama yang ditemukan di direktori proyek.
- Pembersihan Data (Data Cleaning & Prep):
  * Konversi tipe data kolom tanggal (`date`) ke format Datetime.
  * Penanganan nilai kosong (NaN) untuk status promo dan tipe promo.
  * Konversi tipe data kolom numerik secara aman dengan penanganan eror.
  * Pembuatan kolom pembantu (dimensi waktu tahun, bulan, hari libur/weekend, dll.).

4. ALIRAN FILTER REAKTIF & KEADAAN (STATE & FILTER FLOW)
--------------------------------------------------------
- Penyimpanan State: Menggunakan Streamlit Session State (`st.session_state`) untuk mempertahankan pilihan filter pengguna.
- Logika Filter Berjenjang/Kaskade (Cascading Filters):
  * Opsi dropdown untuk setiap filter (Tahun, Cabang, Kota, Tipe Cabang, Promo) dihitung secara dinamis dari subset data yang sedang aktif.
  * Setiap dropdown difilter berdasarkan seluruh pilihan filter LAINNYA untuk mencegah kombinasi filter kosong (no data).
  * Validasi Otomatis: Jika kombinasi filter menjadi tidak valid (misal kota dipindah dari Bandung ke Jakarta saat cabang Dago terpilih), sistem otomatis mereset pilihan yang tidak valid tersebut ke kondisi awal ("All ...").
  * Pengisian Otomatis (Auto-fill): Memilih cabang (Branch) tertentu otomatis mencari kotanya (City) dan tipenya (Branch Type) di database, lalu mengisinya ke dropdown terkait.

5. DESAIN ANTARMUKA & GAYA (UI & CSS LAYER)
-------------------------------------------
- Desain Toko: Menggunakan skema warna bertema kopi (Cokelat & Cream) dengan aksen warna Hijau/Merah khusus untuk indikator perubahan (YoY delta).
- CSS Kustom: Disisipkan menggunakan `st.markdown(..., unsafe_allow_html=True)` untuk:
  * Menyembunyikan menu Streamlit bawaan, header, footer, dan sidebar.
  * Membatasi tinggi viewport secara ketat (`overflow: hidden`) untuk mencegah scrollbar vertikal.
  * Menggunakan font Google Fonts (Inter).
  * Mengatur tata letak (padding, margin, gap) agar rapat dan ringkas.
  * Kustomisasi kartu KPI agar tampil modern dengan bayangan halus, ikon lingkaran, dan badge delta.
  * Responsivitas media query (`@media max-height`) untuk layar laptop yang lebih kecil.

6. STRUKTUR DAN KOMPONEN DASHBOARD
----------------------------------
Aplikasi dibagi menjadi 4 bagian tata letak vertikal:
1. Baris Header & Filter (st.columns):
   - Judul Dashboard & Subjudul.
   - Dropdown Filters: Year, Branch, City, Branch Type, Promo Type.
2. Baris Kartu KPI (4 Kolom):
   - Revenue (Pendapatan total dalam format Rupiah terenkripsi).
   - Total Transactions (Total jumlah transaksi).
   - Avg Ticket Size (Rata-rata pengeluaran per transaksi).
   - Cups Sold (Total jumlah cup kopi yang terjual).
   - YoY Delta Badge: Menampilkan persentase kenaikan/penurunan dibanding tahun sebelumnya (aktif jika filter "Year" dipilih).
3. Baris Tengah (2 Kolom):
   - Performance Overview: Grafik garis ganda (dual-axis) Plotly untuk memantau tren bulanan Revenue (kiri) vs Transactions (kanan).
   - Customer Channel Mix: Grafik donat Plotly untuk melihat persentase kontribusi penjualan via Dine-In, Delivery, dan Takeaway.
4. Baris Bawah (3 Kolom Actionable Insights):
   - Promo vs Non Promo: Grafik batang menegak (Bar) membandingkan rata-rata pendapatan harian saat promo aktif vs non-promo.
   - Top Category Performance: Grafik batang mendatar (Horizontal Bar) yang mengurutkan kategori produk terlaris berdasarkan total pendapatan.
   - Weekday vs Weekend Performance: Grafik batang kelompok (Grouped Bar) membandingkan rata-rata pendapatan, transaksi, dan ticket size pada hari kerja vs akhir pekan.
